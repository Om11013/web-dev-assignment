# Gushwork Assignment: Premium HDPE Pipes & Coils Landing Page

This project is a high-performance, responsive landing page for a premium industrial brand, built exclusively using vanilla HTML, CSS, and JavaScript.

## 🚀 Live Demo & Repository
- **Figma Design Reference:** [Gushwork Assignment Figma](https://www.figma.com/design/DOv07H7C2tA5UrVLhmfwfW/Gushwork-Assignment?node-id=490-8785)
- **Pixel-Perfect Implementation:** Standardized to match the exact design specifications provided.

---

## ✨ Key Features

### 🛠️ Technical Requirements Met:
- **Sticky Header Functionality:**
    - Appears seamlessly when scrolling beyond the first fold.
    - Positions itself elegantly above the navigation bar.
    - Features a smart hide/show logic based on scroll direction (hides on scroll down, reveals on scroll up).
    - Smooth transitions and height-aware animations.
- **Image Carousel with Interactive Zoom:**
    - Custom-built carousel for product showcasing.
    - **Advanced Zoom:** On hovering over the main image, a high-fidelity zoomed preview is displayed with smooth tracking.
    - Thumbnail-based navigation and arrow controls.
- **Responsive Architecture:**
    - Optimized for Desktop (1440px+), Tablet (768px - 1024px), and Mobile (375px - 767px).
    - Flexbox and CSS Grid used for robust layout management.
- **Modern UI Elements:**
    - Custom FAQ Accordions.
    - Sliding Carousels for "Applications" and "Reviews" sections.
    - Dynamic tabs for the "Manufacturing Process" section.
    - Professional Modal Popups for "Download" and "Quote" requests.

---

## 💻 Technologies Used
- **Structure:** Semantic HTML5 for better accessibility and SEO.
- **Style:** Modern CSS3 (Vanilla) with Variables, Flexbox, and Grid.
- **Logic:** Vanilla JavaScript (ES6+) — No external libraries or frameworks (No jQuery, No React).
- **Icons/Assets:** SVG format for crisp visuals at any resolution.

## 🎨 Design Philosophy
The implementation focuses on **Visual Excellence** and **User Experience**.
- **Typography:** Using "Inter" from Google Fonts for a modern, clean look.
- **Aesthetics:** Vibrant accents (Deep Blue/Indigo) paired with soft gradients and glassmorphism-inspired overlays.
- **Micro-interactions:** Hover effects on cards, smooth scrolling, and deliberate animation timing.
